### 1. 🥁 虚拟鼓组 - 30秒教程

```
1. 按键盘 A-K 演奏鼓垫
2. 点击 "⏺ 开始录音"
3. 演奏一段节奏
4. 点击 "⏹ 停止录音"
5. 点击 "💾 导出录音"
6. 选择格式（WebM/WAV/OGG）
7. 文件自动下载！
```

**快捷键:**
- A = Kick (底鼓)
- S = Snare (军鼓)
- D = Hi-Hat (踩镲)
- F-K = 其他鼓垫

---

### 2. 🎵 可视化器 - 30秒教程

```
1. 点击 "选择文件" 上传音乐
   或点击 "生成音乐" 使用合成音
2. 点击 "▶ 播放"
3. 切换可视化模式（Bars/Dots/Grid等）
4. 调整参数（速度/密度/色相/亮度）
5. 享受视觉盛宴！
```

**提示:** 拖动进度条可跳转到任意位置

---

### 3. 🎚️ 音频效果器 - 30秒教程

```
1. 上传音频文件 或 开启麦克风
2. 点击 "▶ 播放"
3. 调整均衡器（低/中/高频）
4. 开启混响、延迟等效果
5. 实时听到效果变化
```

**推荐顺序:** EQ → 失真 → 滤波器 → 混响/延迟

---

### 4. 🎹 虚拟钢琴 - 30秒教程

```
1. 选择音色（钢琴/电钢琴/风琴/合成器）
2. 按键盘 A-K 演奏
3. 调整 ADSR 参数
4. 使用 +/- 切换八度
5. 开启"持续音"创作和弦
```

**快捷键:** A-K = 低八度, K-B = 高八度

## 🔧 常见问题

### Q: 没有声音？
**A:** 
1. 检查系统音量
2. 检查浏览器是否静音
3. 尝试点击其他按钮（浏览器需要用户交互）

### Q: 麦克风无法使用？
**A:**
1. 浏览器提示时点击"允许"
2. 检查系统麦克风权限
3. 使用Chrome或Firefox

### Q: 导出的文件打不开？
**A:**
1. WebM需要支持的播放器（VLC, Chrome）
2. 尝试导出WAV格式（通用）
3. Safari用户建议使用WAV

### Q: 性能卡顿？
**A:**
1. 关闭浏览器其他标签页
2. 使用Chrome或Edge（性能最佳）
3. 降低可视化参数（速度、密度）

## 💡 使用技巧

### 提升音质
1. 使用均衡器微调频率
2. 适度添加混响（30-50%）
3. 使用滤波器塑造音色

### 创作流程
1. 先从节奏开始（鼓组）
2. 添加旋律（钢琴）
3. 后期处理（效果器）
4. 可视化展示

### 键盘演奏
1. 调整Release参数（更自然）
2. 使用持续音模式（和弦）
3. 熟悉键位映射（提高速度）
